# Backend Ledger API

A Node.js + Express based Banking Ledger Backend System.

## Features

* JWT Authentication
* User Registration & Login
* Ledger Transaction Management
* MongoDB Database Integration
* Email Notification Support

## Tech Stack

* Node.js
* Express.js
* MongoDB (Mongoose)
* JWT Authentication
* Nodemailer

## Project Setup

Clone the repository:

git clone https://github.com/ashwani-codess/backend-ledger.git

Install dependencies:

npm install

Create `.env` file:

MONGO_URI=
JWT_SECRET=

Run Server:

npm run dev
